# C++ Programming Fundamentals: Practical Exercises

This repository contains a collection of basic C++ applications and problem-solving exercises. It is designed to document my journey of learning programming from scratch, focusing on mastering the core syntax and logic of the C++ language.

## 📌 Project Overview
The project covers the fundamental building blocks of C++, starting from simple console interactions to implementing logic that simulates built-in system functions. The main goal is to build a strong foundation in procedural programming and logical thinking.

## 🛠️ Core Concepts Covered
The following C++ tools and features are implemented and practiced within this project:

* **Basic I/O & Variables:** Using `cin`, `cout`, and various data types.
* **Arithmetic Operators:** Performing mathematical calculations and data manipulation.
* **Control Flow:**
    * Conditional statements: `if`, `if-else`, and `else`.
    * Switch-case statements for multi-way branching.
* **Loops:**
    * `for` loops and `nested for loops` for pattern printing and data processing.
    * `while` and `do-while` loops.
* **Data Grouping:**
    * **Structs:** Defining custom data structures.
    * **Enums:** Creating user-defined types for better code readability.
* **Arrays:** Handling collections of data.
* **Functions:** Modularizing code and understanding scope.
* **Randomness:** Utilizing built-in functions to generate random numbers.

## 🚀 Key Learning Milestones

### 1. Tool Integration
Practical training on combining multiple concepts to solve more complex problems, such as:
* Integrating **Enums** with **Structs** and **Functions** to model real-world data.
* Using **Switch cases** with **Enums** to create clean, readable menu-driven logic.

### 2. Math Library Simulation
To deeply understand how standard libraries work, I have manually implemented logic to simulate several built-in C++ math functions without using `<cmath>`:
* `round()`: Custom rounding logic.
* `floor()` & `ceil()`: Understanding integer boundaries.
* `abs()`: Implementing absolute value logic.
* `pow()`: Creating power functions using loops.
* `sqrt()`: Approximating square roots through logic.

### 3. Problem Solving
Solved various "beginner-friendly" logic puzzles to practice the syntax, such as calculating grades, simple banking simulations, and basic geometric calculations.
